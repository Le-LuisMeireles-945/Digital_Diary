One of the games that fits most if not all the categories discussed in class (travel to a new place, deal with a different character, and doing something inaccessible in real life), was Undertale. Undertale is a very narrative driven game, in which it begins by our character felling from a tunnel into a new world, the first thing the main character encounters is a cute looking follower, that tries to trick the player in order to kill him. The game is filled with absurd over the top monsters, that in contrary to other RPG’s get the chance to either fight or befriend and forgive them.

PT
Um dos jogos que, na minha opinião, se enquadra com os tópicos que foram pedidos em aula (viajar a um sítio novo, encarnar uma personagem diferente, fazer algo inacessível na vida real) é sem dúvida o Undertale. Undertale é um jogo que é muito focado na sua história, na qual começa com a nossa personagem a cair de uma toca para um mundo novo. O primeiro encontro que temos como personagem é uma florzita que tenta enganar o jogador e matá-lo. O jogo está cheio de monstros ridículos e ao contrário de outros RPG’s podemos lutar ou perdoare-los.

Unity

ENG
During the development of Unitys first game kit, we were able to thinker with what the framework was to offer. Having some previous knowledge, due to previous classes, of how Unity works this step was relatively simple to me. The first kit was a simple Kart game, the first sept the tutorial of the game kit instructed was to change the players kart color. We could do that by simply double clicking the kart on the scene or finding it on the main scene hierarchy on the left. After selecting the object, we need to navigate on the project folder and go to “Assets/Materials/Kart_Classic”, and select the “KartBody” asset. Here we can change the color of this asset on the menu represented on the image bellow, selecting the Albedo option with the value that is pretended.

PT
Durante o desenvolvimento em Unity, do primero kit, foi onde mexemos melhor com as ferramentas que a framework oferece. Tendo já algum conhecimento prévio da ferramenta em outras cadeiras, foi-me relativamente fácil compreender a parte da navegação em Unity. O primeiro kit com qual se trabalhou foi um jogo de Karts, simples, e seguimos o tutorial disponível pelo unity. A primeira etapa que foi feita, foi mudar a cor do Kart, onde simplesmente selecionávamos com um duplo click no componente ou através da “MainScene”. Com o objeto selecionado, podemos ir a pasta “Assets/Materials/Kart_Classic” do projeto e selecionar “KartBody”. Podemos alterar a cor deste asset no menu representado na imagem alterando o Albedo para o valor que pretendemos:

ENG
The game kit provides us with a way to add a ramp to the karting track, and by adding this asset it makes it that the level becomes more interesting and customizable. We are able to find this asset in the following path in the project folder “Assets/Karting/Prefabs/Props”.

PT
Por sequente, algo que também é fornecido pelo kit é a opção de podermos adicionar rampas a pista. Tornando o percurso mais customizavel, sendo que podemos encontrar o asset da rampa no seguinte caminho “Assets/Karting/Prefabs/Props”.

ENG
To conclude the last script, I followed from the tutorial without entering level design, was to add a turbo zone asset. We can edit this asset to add more velocity to the kart, in that given zone, but we should pay attention that as fun as it is to abuse the velocity value we should avoid doing so, since it could break the wall collider on the tracks. To add this asset, we can find it on the path “Assets/Mod Assets/Kart” and drag it to the right position on the map.

PT
Por último a última, coisa que fiz com este kit sem entrar em level design, foi adicionar uma zona de turbo. Neste asset é possível a edição da velocidade que o kart ganha quando está a passar por aquela zona, é de notar que por muito engraçado que seja dar um valor absurdo a velocidade ganha pelo kart, muito facilmente é possível sair do collider da pista. Para adicionar este asset basta ir ao caminho “Assets/Mod Assets/Kart” e arrastar para a respetiva posição do mapa.
